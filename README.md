Este é o meu portfólio profissional, reconstruído para focar em **alta performance**, **design minimalista** e **experiência do usuário**. O projeto demonstra minhas habilidades em criar sistemas robustos com interfaces impecáveis, utilizando o que há de mais moderno no ecossistema web.

---

## ✨ O que há de novo?

- **Animações Dinâmicas**: Uso da biblioteca **AOS** (Animate On Scroll) para uma navegação fluida e moderna.
- **Comunicação Profissional**: Integração com **EmailJS**, permitindo o recebimento de mensagens diretamente no e-mail sem recarregar a página.
- **Feedback Visual**: Alertas personalizados com **SweetAlert2** para confirmações de envio de formulário.
- **Navegação Inteligente**: Menu fixo com lógica de scroll e suporte a dispositivos móveis (Mobile-first).
- **Design Moderno**: Estilização via **Tailwind CSS** (v4) com foco em tipografia e contraste.

---

## 🛠️ Stack Tecnológica

### **Front-end**
- **HTML5 & JavaScript (ES6+)**: Lógica e estrutura.
- **Tailwind CSS**: Estilização utilitária de alta performance.
- **AOS.js**: Efeitos de entrada de elementos no scroll.
- **DevIcons**: Identidade visual das tecnologias utilizadas.

### **Integrações**
- **EmailJS**: Serviço de envio de e-mails via client-side.
- **SweetAlert2**: Pop-ups de interface refinada.
